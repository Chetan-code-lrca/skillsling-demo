# SkillSling - Offline AI Tutor

An offline AI-powered tutor app for students (Hindi/English support).
Runs locally using Ollama + Streamlit + gemma3:4b model.

## How to Run Locally

1. Install Ollama: https://ollama.com
2. Pull model: ollama run gemma3:4b
3. Install dependencies: pip install streamlit ollama
4. Run: streamlit run app.py
